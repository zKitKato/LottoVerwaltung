# Spieler-Verwaltung

Die Spieler-Verwaltung stellt das zentrale Modul der Applikation dar. In diesem Bereich können Sie bestehende Datensätze
einsehen, filtern, bearbeiten und neue Teilnehmer erfassen.

---

## Spielerübersicht und Suche

In der zentralen Übersicht werden alle registrierten Teilnehmer in einer tabellarischen Ansicht (Player-Table)
dargestellt.

![Spielerübersicht](/img/SpielerUebersicht.png)

### Suchfunktion

Um gezielt nach einzelnen Teilnehmern zu suchen, nutzen Sie das Suchfeld oberhalb der Tabelle.

![Spielersuche](/img/SpielerSuche.png)

* **Funktionsweise**: Während der Eingabe wird die Tabelle in Echtzeit gefiltert. Sie können nach Namen, Usernames oder
  anderen Identifikatoren suchen, um die Auswahl der angezeigten Datensätze einzugrenzen.

### Interaktive Sortierung

Die Kopfzeile der Tabelle ist interaktiv gestaltet. Sie können auf die folgenden Bezeichnungen klicken, um die Liste
wahlweise aufsteigend oder absteigend zu sortieren:

* **Username**, **Spielt seit**, **Spiele**, **Kontostand** und **Status**.

---

## Spieler hinzufügen

Um einen neuen Teilnehmer im System zu registrieren, nutzen Sie die Schaltfläche "Spieler hinzufügen".

![Spieler hinzufügen](/img/SpielerHinzufuegen.png)

### Vorgehensweise bei der Neuanlage

1. **Datenaufnahme**: Geben Sie den Benutzernamen sowie alle weiteren erforderlichen Informationen ein.
2. **Validierung**: Das System führt eine automatische Prüfung auf Vollständigkeit durch.
3. **Abschluss**: Schließen Sie den Vorgang durch Klicken auf die Schaltfläche **"Speichern"** ab.

---

## Spieler bearbeiten

Sollten sich Stammdaten eines Spielers ändern oder Korrekturen am Kontostand notwendig sein, nutzen Sie die
Bearbeitungsfunktion. Diese rufen Sie in der Regel über ein Edit-Icon innerhalb der jeweiligen Zeile der
Spielerübersicht auf.

![Spieler bearbeiten](/img/SpielerBearbeiten.png)

### Anpassung der Datensätze

1. **Maske öffnen**: Nach Auswahl des Spielers öffnet sich die Eingabemaske mit den aktuell hinterlegten Daten.
2. **Änderungen vornehmen**: Passen Sie die gewünschten Felder (z. B. Status oder persönliche Informationen) an.
3. **Aktualisierung**: Bestätigen Sie die Änderungen durch Klicken auf die Schaltfläche **"Speichern"** oder **"
   Aktualisieren"**. Die Datenbank wird daraufhin unmittelbar angepasst.

---

## Spielerprofil

Das Spielerprofil bietet eine detaillierte Einzelansicht eines Teilnehmers. Hier können spezifische Statistiken, die
Historie der Tippscheine sowie persönliche Einstellungen eingesehen werden.

![Spielerprofil](/img/SpielerProfil.png)

### Zugriff auf das Profil

Sie haben zwei Möglichkeiten, das Profil eines Spielers aufzurufen:

1. **Via Home-Menü**: Nutzen Sie die zentrale Suchfunktion auf der Startseite. Sobald Sie den Namen des Spielers
   eingeben und den entsprechenden Datensatz auswählen, werden Sie direkt zum Profil weitergeleitet.
2. **Via Spielerübersicht**: Innerhalb der Player-Table ist der **Username** als interaktiver Link gestaltet. Ein
   einfacher Klick auf den Namen des gewünschten Teilnehmers öffnet dessen Profilansicht.

In dieser Ansicht lassen sich alle individuellen Daten des Spielers zentral verwalten, was eine gezielte Administration
ermöglicht.

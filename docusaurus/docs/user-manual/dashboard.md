# Dashboard und globale Navigation

Das Dashboard dient als zentrale Schaltzentrale der Applikation. Von hier aus haben Sie Zugriff auf alle Untermodule und
können systemweite Suchen durchführen.

![Dashboard Übersicht](/img/DashboardUebersicht.png)

---

## Navigation über die Sidebar

Auf der linken Seite befindet sich die Sidebar, über die Sie schnell zwischen den verschiedenen Funktionsbereichen
wechseln können. Folgende Seiten sind darüber erreichbar:

* **Spieler-Verwaltung**: Verwaltung der Teilnehmerdaten und Profile.
* **Ticket-Verwaltung**: Übersicht und Erstellung von Tippscheinen.
* **Add-ons (Ziehungen)**:
    * **Lotto-Ziehungen**: Verwaltung und Einsicht der Ergebnisse von 6aus49.
    * **Euro-Ziehungen**: Verwaltung und Einsicht der Eurojackpot-Ergebnisse.

---

## Systemweite Suche

Das Dashboard verfügt über eine intelligente Suchleiste, die Ihnen hilft, Informationen schnell aufzufinden.

![Dashboard Suche](/img/DashboardSuche.png)

### Funktionsumfang der Suche

Die Suchleiste ist so konzipiert, dass sie verschiedene Datentypen gleichzeitig verarbeiten kann:

* **Seitensuche**: Geben Sie den Namen eines Moduls ein (z. B. "Tickets"), um direkt dorthin zu navigieren.
* **Spielersuche**: Suchen Sie nach Namen oder Usernames, um direkt zum jeweiligen Spielerprofil zu gelangen.

---

## Benutzerkonto und Abmeldung

In der oberen rechten Ecke der Benutzeroberfläche finden Sie Ihr rundes Profil-Icon. Dieses Element dient der Verwaltung
Ihrer aktuellen Sitzung.

* **Logout**: Durch einen Klick auf das Profil-Icon öffnet sich ein Dropdown-Menü. Dort finden Sie die Option **Logout
  **, um sich sicher von der Applikation abzumelden und die Sitzung zu beenden.
